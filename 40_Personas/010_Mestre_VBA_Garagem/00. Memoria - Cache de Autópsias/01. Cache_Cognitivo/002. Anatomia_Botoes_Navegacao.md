# Nota Técnica: Anatomia dos Botões de Navegação (V3)

**Data**: 14/01/2026
**Assunto**: Remapeamento de Botões de Navegação Interna

### O Padrão Legado
Os botões de navegação lateral (Shapes) nas planilhas seguem a nomenclatura:
*   **PG1**: Mecânica / Estrutura
*   **PG2**: Elétrica
*   **PG3**: IHM

Macros associadas no CCMX: `CCMXPG1`, `CCMXPG2`, `CCMXPG3`.

### Técnica de Reparo Automático
Para evitar reatribuição manual (que pode ser bloqueada por restrições de interface), deve-se usar um script de loop:
```vba
For Each shp In ActiveSheet.Shapes
    If shp.OnAction Like "*CCMXPG1*" Then shp.OnAction = "PLTPG1"
    ' ... Repetir para PG2 e PG3
Next shp
```

### Aviso para Futuros Resets
Se o usuário reclamar que "clica no botão e volta para o CCMX", o script de reparo esqueceu de tratar um dos IDs de página. Verifique o `.OnAction` do shape via `Debug.Print`.
