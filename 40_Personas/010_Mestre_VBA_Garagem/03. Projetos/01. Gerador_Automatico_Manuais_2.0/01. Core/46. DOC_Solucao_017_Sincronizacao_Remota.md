# ✅ Solução 017: Comando de Sincronização Remota

> **Origem**: [[45. DOC_Demanda_017_Sincronizacao_Remota|Demanda 017]]

## 1. O Problema Resolvido
Contorno do travamento de autenticação do Git em ambiente CLI restrito.

## 2. A Solução (Extração Cirúrgica via Subtree)
Para enviar **apenas** a pasta do Gerador V2 e ignorar os 814MB do restante do LabCogKHS, use este comando no CMD (estando na raiz `C:\LabCogKHS_CLI`):

### Comando de Extração:
```cmd
git subtree push --prefix="40_Personas/040. Mestre em VBA (A Garagem de Autópsias Digitais)/03. Projetos/01. Gerador_Automatico_Manuais_2.0" https://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@github.com/RicardoCabralBanci/00.-Tarde-de-Domingo-.git master
```

*Nota: Esse comando vai processar apenas os arquivos da pasta V2. Será ordens de magnitude mais rápido.*

## 3. Alertas de Segurança
- **Token Exposto**: Este arquivo contém um token sensível. Em ambiente de produção real, este arquivo deve ser ignorado pelo Git ou o token deve ser revogado após o uso.
- **Histórico**: Após rodar o comando, limpe o histórico do terminal.
