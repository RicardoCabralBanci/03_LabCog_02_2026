*   [ ] **Renderização**: Verificar se `Shapes.AddPicture` está respeitando o `LockAspectRatio` para não distorcer fotos de IHMs widescreen em cards quadrados.

**Sua Verificação Manual**:

## 2. O Desafio da Zona B (Ícone do Word)
**Meta**: Substituir o desenho vetorial tosco pelo ícone `icon_word.png` localizado em `assets\System\`.

**Tarefas de Código (`mod_UI_ZoneB`)**:
*   [ ] **Implementar Função de Ícone**: Reintroduzir a lógica que tenta carregar `assets\System\icon_word.png`.
*   [ ] **Fallback Seguro**: Se o PNG sumir, o código DEVE desenhar o vetor antigo automaticamente. O sistema não pode depender de um arquivo externo para funcionar (robustez).
*   [ ] **Transparência**: Garantir que o PNG seja inserido com `msoFalse, msoTrue` (LinkToFile = False, SaveWithDocument = True) para que a transparência funcione e o ícone não suma se a planilha for enviada por e-mail.
