# Planejamento POC: Preenchimento de Tabela OpenXML

**Objetivo**: Validar a técnica de clonagem de linhas (Row Cloning) para preencher tabelas do Word mantendo a formatação original, sem usar OLE/Clipboard.

## Cenário de Teste
1.  **Input**:
    - Arquivo `Template.docx` contendo uma tabela vazia (1 linha de cabeçalho + 1 linha de dados vazia) com Alt Text = "Componentes integrados".
    - Dados simulados:
        - Linha 1: "Esteira", "KHS", "123"
        - Linha 2: "Rotuladora", "Krones", "456"
        - Linha 3: "Paletizadora", "Sidel", "789"

2.  **Processo (Código C#)**:
    - Abrir `Template.docx`.
    - Localizar tabela por `TableProperties.TableCaption` == "Componentes integrados".
    - Identificar a "Linha Molde" (última linha da tabela ou uma linha específica).
    - Para cada item de dados:
        - Clonar a "Linha Molde".
        - Substituir os textos das células (`TableCell`).
        - Adicionar a nova linha à tabela.
    - (Opcional) Remover a linha molde original se ela estava vazia.
    - Salvar como `Resultado.docx`.

3.  **Critérios de Sucesso**:
    - O arquivo `Resultado.docx` abre sem erros.
    - A tabela contém 3 linhas de dados + cabeçalho.
    - As bordas, fontes e alinhamentos das novas linhas são idênticos aos da linha original.
