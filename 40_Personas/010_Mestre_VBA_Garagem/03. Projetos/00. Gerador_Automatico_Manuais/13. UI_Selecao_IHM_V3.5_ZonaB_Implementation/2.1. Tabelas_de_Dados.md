### Notas Técnicas para Implementação (VBA)

1.  **Relacionamento**: A coluna `IHM_ID` na Tabela 2 é uma *Foreign Key* para a coluna `ID` na Tabela 1.
2.  **Caminhos Relativos**: O código VBA deve concatenar `ThisWorkbook.Path` com os caminhos listados (ex: `templates/v3/...`) para evitar links quebrados ao mover o projeto.
3.  **Flexibilidade**: A Tabela 2 permite que IHMs diferentes reutilizem o mesmo arquivo Word se necessário, ou tenham estruturas completamente diferentes (ex: IHM 6 só tem capítulos 1 e 4).
