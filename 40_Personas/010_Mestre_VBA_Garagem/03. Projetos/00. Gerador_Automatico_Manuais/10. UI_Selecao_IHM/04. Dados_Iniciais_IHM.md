## Notas de Implementação no Excel
1. **Nome da Tabela**: `tbl_IHM_Models` (Essencial para o C# encontrar o Range via `ListObjects`).
2. **Localização**: Aba `sys_data` (VeryHidden).
3. **ID_KEY**: Esta coluna deve ser única e imutável. É ela que o código usará para salvar a preferência do usuário.
