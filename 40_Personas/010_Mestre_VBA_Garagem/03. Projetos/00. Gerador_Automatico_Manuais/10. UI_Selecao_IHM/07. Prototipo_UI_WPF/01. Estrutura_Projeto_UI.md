# Estrutura Técnica: UI de Seleção de IHM (WPF)

**Tecnologia**: C# / WPF (.NET 6.0+)  
**Padrão**: MVVM (Model-View-ViewModel)  
**Objetivo**: Criar uma janela fluida com grade de Cards para seleção de IHM.

---

## 1. O Modelo de Dados (`IHMModel.cs`)
Esta classe será o espelho da nossa tabela do Excel/CSV.
```csharp
public class IHMModel {
    public int Id { get; set; }
    public string Codigo { get; set; }
    public string Fabricante { get; set; }
    public string Modelo { get; set; }
    public string ThumbnailPath { get; set; }
    public string Descricao { get; set; }
}
```

---

## 2. A Camada Visual (XAML)
Usaremos um `ItemsControl` (ou `ListBox`) com um `ItemTemplate` customizado.
*   **Card Background**: Bordas arredondadas (`CornerRadius="8"`).
*   **Shadows**: `DropShadowEffect` para dar profundidade.
*   **Interatividade**: Mudança de cor no `IsMouseOver`.

---

## 3. O "Pulo do Gato": Integração Excel
O ViewModel terá um método `LoadDataFromExcel()` que:
1.  Abre a conexão via Interop (ou lê o CSV se estivermos com pressa).
2.  Popula uma `ObservableCollection<IHMModel>`.
3.  Faz o Binding automático com a tela.

---

## 4. Próximos Passos
1. Criar o protótipo do **XAML** (A casca).
2. Criar a lógica do **ViewModel** (O cérebro).
3. Implementar o evento de **Clique no Card** que injeta o ID no Excel.

**Nota do Mestre**: Se você quiser que isso rode dentro do Excel como um Add-in, o código é 90% o mesmo, só muda o jeito que a janela é chamada.
