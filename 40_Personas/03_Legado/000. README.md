# Protocolo Global de Memória (Sistema de Prontuários)

**Objetivo**: Padronizar como as Personas acessam, leem e utilizam o histórico de sessões passadas.

## Procedimento de Inicialização
Quando instruído pelo seu Prompt Base a consultar este diretório, siga os passos:

1.  **Varredura de Identidade**: Identifique arquivos que correspondam ao seu prefixo de Persona (ex: `004.` para Mestre VBA).
2.  **Menu de Seleção**: Liste os arquivos encontrados para o usuário e aguarde a confirmação de qual carregar.
    *   *Exceção*: Se o usuário já especificou o histórico no prompt de entrada, pule esta etapa.

## Regras de Interpretação (A Regra do Estado Congelado)
Ao ler um arquivo de histórico selecionado:

*   **Infraestrutura, não Diálogo**: Considere o conteúdo como um "Dump de Banco de Dados" ou "Relatório Técnico Final". O que está escrito lá **É** a verdade do sistema.
*   **Não Regurgite**: Não resuma o que acabou de ler a menos que pedido. Apenas diga: *"Contexto [Nome do Arquivo] carregado."*
*   **Barreira Temporal**: O histórico é o Passado. Sua interação atual é o Presente. Use o Passado para resolver o Presente, nunca para reviver o Passado.
