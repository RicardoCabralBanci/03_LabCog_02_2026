# Detalhamento: Zona C (Landing Zone & Ignite)

**Localização**: Colunas **M até P**.
**Função**: Captura da imagem de capa (Screenshot) e execução do motor de geração.

---

## 1. A Landing Zone (M5:P19)

O "berço" para o print da IHM. Deve ser visualmente distinta para convidar a interação.

*   **Design do Container**:
    *   **Fundo**: Preenchimento Cinza muito claro ou transparente.
    *   **Borda**: Estilo tracejado (`DashStyle: Dash`), Cor Cinza Médio, Espessura 2pt.
    *   **Texto Central (Placeholder)**: "COLE O PRINT DA IHM AQUI (CTRL+V)" em fonte grande, cinza claro, centralizado no range.

*   **Comportamento de Colagem (`UI_Paste_Smart`)**:
    1.  O VBA detecta a entrada de um novo Shape ou o clique no botão "Importar".
    2.  **Fit Logic**: A imagem é redimensionada para caber no Range M5:P19 mantendo a proporção (`LockAspectRatio = True`).
    3.  **Naming**: O shape é renomeado para `IMG_USER_COVER`.
    4.  **Feedback Visual**: Ao colar com sucesso, a borda tracejada muda para **Verde Sólido** e o placeholder desaparece.

---

## 2. Ferramentas e Gatilhos

### A. Botão [Importar Imagem] (Linha 21)
*   **Ação**: Abre o diálogo `Application.GetOpenFilename` para buscar um arquivo local (fallback caso o Ctrl+V falhe).
*   **Estilo**: Botão discreto, cinza médio.

### B. O Botão "GERAR MANUAL" (Linha 24, Colunas M:P)
O botão mais importante da interface.
*   **Design**:
    *   **Largura**: Ocupa de M até P.
    *   **Cor**: Azul KHS (ou Verde Ação) - a única cor vibrante fora das marcas das IHMs.
    *   **Texto**: "GERAR MANUAL" (Fonte Branca, Bold, 12pt).
*   **Lógica do Clique (`Main_Ignite_Sequence`)**:
    1.  **Validação**: Verifica se `IMG_USER_COVER` existe. Se não existir, exibe aviso: "Nenhuma imagem de capa detectada. Deseja usar a imagem padrão?".
    2.  **Preparação**: Exporta o print para uma pasta temporária.
    3.  **Execução**: Envia a lista de capítulos (Zona B) e o caminho da imagem para o motor de geração (C# ou VBA Word).

---

## 3. Lógica de Reset
*   Ao trocar de IHM na Zona A, a Zona C **não limpa** o print automaticamente (pode ser o mesmo print para variantes do mesmo modelo), mas exibe uma pequena notificação sugerindo a troca da imagem se o modelo for muito diferente.
