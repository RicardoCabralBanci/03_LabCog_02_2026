# Ephemeris

## Definição
Uma **efeméride** (do grego *ephemeris*, "diário"; plural: *efemérides*) é uma tabela, livreto ou recurso digital que fornece informações detalhadas sobre as posições e, por vezes, velocidades de corpos celestes (como planetas, luas, estrelas, asteroides e satélites artificiais) em momentos específicos. Historicamente eram tabelas impressas, mas hoje são frequentemente disponibilizadas em formatos eletrônicos.

## Uso em Astronomia
*   **Rastreamento e Previsão:** Usado para rastrear o movimento de objetos celestes naturais e satélites artificiais, prevendo suas posições precisas.
*   **Navegação Celestial:** Vital como backup para sistemas de navegação por satélite.
*   **Pesquisa Científica:** Sistemas sofisticados (como os do JPL da NASA) geram posições altamente precisas para pesquisa, cobrindo séculos no passado e futuro.
*   **Astronomia de Pulsares:** Arquivos de efemérides contêm parâmetros essenciais para modelos de temporização de pulsares.

## Uso em Software
*   **Formatos Digitais:** Componente central de muitas ferramentas de software astronômico.
*   **Bibliotecas de Software:** Bibliotecas em C, Java, Fortran, etc., são projetadas para ler/escrever arquivos de efemérides e realizar cálculos de alta precisão.
*   **Sistemas GPS:** O sinal GPS incorpora dados de efemérides para calcular a posição precisa dos satélites em órbita.
*   **Software de Astrologia:** Utilizado para interpretar movimentos planetários e gerar previsões.
