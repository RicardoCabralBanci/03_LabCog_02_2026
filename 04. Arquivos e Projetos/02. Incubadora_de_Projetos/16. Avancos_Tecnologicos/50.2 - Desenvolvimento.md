# 50.2 - Desenvolvimento de Software

## Estado da Arte Atual
- **AI Coding Agents:** Devin, GitHub Copilot Workspace, Cursor. O foco muda de "escrever código" para "arquitetar soluções" e "revisar código gerado".
- **Self-Healing Code:** Arquiteturas que detectam falhas e tentam correções autônomas.
- **Linguagens:** Rust ganhando tração no Kernel e infraestrutura crítica; Python dominando IA; TypeScript onipresente na Web.

## Discussão
Acesso a ferramentas de nível sênior para juniores. O risco de "code bloat" gerado por IA e a perda de fundamentos de computação.
