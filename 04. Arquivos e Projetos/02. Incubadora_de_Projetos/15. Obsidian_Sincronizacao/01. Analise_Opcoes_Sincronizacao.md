# Estratégias de Sincronização do Obsidian

Este documento analisa as opções para sincronizar um cofre (Vault) do Obsidian entre PC (Windows) e Tablet.

## Tabela Comparativa

| Método                   | Custo             | Dificuldade Técnica | Privacidade          | Estabilidade | Notas                                                                                                                     |
| :----------------------- | :---------------- | :------------------ | :------------------- | :----------- | :------------------------------------------------------------------------------------------------------------------------ |
| **Obsidian Sync**        | Pago (~$8/mês)    | Muito Baixa         | Alta (E2E)           | Excelente    | Solução oficial. Funciona em tudo (iOS, Android, Windows, Linux).                                                         |
| **Git (GitHub/GitLab)**  | Grátis            | Alta                | Média/Alta (Privado) | Boa          | Requer conhecimentos de Git. Ótimo para versionamento. Requer plugin "Obsidian Git".                                      |
| **Syncthing**            | Grátis            | Média               | Alta (P2P)           | Excelente    | Sincroniza diretamente device-to-device. **Não funciona em iOS/iPadOS**.                                                  |
| **Remotely Save**        | Grátis/Barato     | Média               | Depende do S3/WebDAV | Boa          | Plugin comunitário. Usa S3, Dropbox, OneDrive ou WebDAV.                                                                  |
| **iCloud**               | Grátis (+storage) | Baixa               | Média (Apple)        | Variável     | Padrão para iPad/Mac. No Windows, o cliente do iCloud pode ser lento/instável com muitos arquivos pequenos.               |
| **FolderSync (Android)** | Grátis/Barato     | Média               | Média                | Boa          | Usa Google Drive/Dropbox/OneDrive como "ponte". O app FolderSync no Android baixa os arquivos para o armazenamento local. |

## Detalhamento das Opções

### 1. Obsidian Sync (Oficial)
A forma "pague para não ter dor de cabeça".
- **Prós:** Configuração em 1 minuto, histórico de versões de 1 ano, criptografia ponta-a-ponta.
- **Contras:** Custo recorrente.

### 2. Obsidian Git
Transforma seu cofre em um repositório Git.
- **Prós:** Backup robusto, controle de versão linha a linha, gratuito.
- **Contras:** Configurar chaves SSH no tablet pode ser chato. Conflitos de merge se editar nos dois lugares ao mesmo tempo sem sincronizar.

### 3. Syncthing (Rei do Android)
Mantém pastas sincronizadas entre dispositivos na mesma rede (ou via relays) sem nuvem central.
- **Prós:** Rápido, privado, gratuito.
- **Contras:** Sem suporte oficial iOS. Precisa que ambos os dispositivos estejam ligados ocasionalmente para trocar dados.

### 4. Remotely Save (Plugin)
Alternativa ao Sync oficial usando nuvens pessoais.
- **Prós:** Flexível (suporta S3, Dropbox).
- **Contras:** Depende de plugin de terceiro. Pode ter conflitos se não fechar o app corretamente.
