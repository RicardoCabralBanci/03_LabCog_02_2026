# Procedimento 01: Criação do Bloco Paletizador no Excel (VBA)

Este guia orienta a implementação completa de 5 abas para o módulo Paletizador.

## Passo 1: Clonagem das Abas
Copie as abas do CCMX na ordem correta e renomeie o **CodeName** (Name) no VBA (F4).

1. **Planilha43**: Copiar da Planilha38 (CCMX Estrutura)
2. **Planilha44**: Copiar da Planilha39 (CCMX Componentes Gerais)
3. **Planilha45**: Copiar da Planilha40 (CCMX Dados Técnicos)
4. **Planilha46**: Copiar da Planilha41 (CCMX Elétrica)
5. **Planilha47**: Copiar da Planilha42 (CCMX IHM)

## Passo 2: Injeção do Código VBA
Substitua a rotina no módulo **Ribbon**:

```vba
'Callback for IHM3 onAction - REDIRECIONADO PARA PALETIZADOR IHM (47)
Sub IHM3(control As IRibbonControl)
    On Error GoTo ErroPaletizador
    Planilha47.Activate 
    MsgBox "Interface do Paletizador (47) carregada.", vbInformation, "KHS"
    Exit Sub
ErroPaletizador:
    MsgBox "Erro: Aba Planilha47 não encontrada.", vbCritical
End Sub
```
